# Outputs

The script generates:
- HTML technical report
- log file
- optional JSON payload snapshot
- optional action queue rows in `DBA_Observability.dbo.AgentActions`

## Output file patterns
- Report: `DBA_Agent_Report_Technical_v4_YYYYMMDD_HHMMSS.html`
- Log: `DBA_Agent_YYYYMMDD_HHMMSS.log`
- Payload: `DBA_Agent_Payload_YYYYMMDD_HHMMSS.json`
